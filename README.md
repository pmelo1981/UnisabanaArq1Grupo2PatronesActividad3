# Product API - Microservicio REST
API REST para gestion de productos

## Tecnologias
.NET 10, Docker, Kubernetes/AKS, Helm, GitHub Actions, ArgoCD

## API
GET /api/products
GET /api/products/{id}
POST /api/products
PUT /api/products/{id}
DELETE /api/products/{id}
GET /api/products/health
